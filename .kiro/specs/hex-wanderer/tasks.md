# 实施计划：HexWanderer（六边形浪游者）

## 概述

基于纯 ES6+ 模块 + CDN PixiJS 的六边形 Roguelike 游戏。自底向上实施：核心工具 → 数据层 → 游戏逻辑 → 渲染层 → UI → 整合。每个核心模块附带单元测试，使用自定义浏览器端测试运行器（不引入 npm）。
