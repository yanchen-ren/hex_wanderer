# 实施计划 v1.2：建筑扩展、事件系统重构、新怪物

## 概述

基于 v1.1，大规模扩展建筑系统（18个建筑）、重构事件系统（200+事件，地形限制，过夜事件）、新增60+战斗怪物、UI改进。

## 阶段一：基础架构扩展

- [x] 1. 事件系统架构升级
  - [x] 1.1 事件配置新增 allowedTerrains 字段，MapGenerator 放置事件时检查地形匹配
  - [x] 1.2 新增 result type：teleport_random（随机传送）、teleport_building（传送到指定建筑）、consume_item（消耗道具）、remove_status（移除debuff）、vision_permanent（永久视野）、reset_combat_events（重置战斗事件）、fog_reveal_random（随机揭雾）、trade（金币交易）、sacrifice_item（献祭道具）
  - [x] 1.3 新增 condition type：has_metal_item（持有金属道具）、has_item_quality（持有指定品质道具）、on_terrain（地形检查）、hp_above（HP阈值）
  - [x] 1.4 事件结果对话框优化：无收益选择（离开/放弃/绕路）不弹结束对话框
  - [x] 1.5 事件密度从15%提高到30%+

- [x] 2. 过夜事件系统
  - [x] 2.1 TurnSystem 回合结束时新增过夜事件触发逻辑
  - [x] 2.2 过夜事件按优先级触发：建筑事件 > 道具事件 > 地形事件 > 通用事件
  - [x] 2.3 过夜事件受地形、建筑、道具三重影响
  - [x] 2.4 实现通用过夜事件：失眠、生病、半夜不死族、半夜强盗
  - [x] 2.5 实现地形过夜事件：狼群夜袭、星空观测（含圣物位置提示）、夜间沙暴、夜间商人（含黑心商人）、半夜毒气、夜间风暴、精灵之歌、女妖之歌、夜间暴风雪、熔岩震动
  - [x] 2.6 实现道具过夜事件：铲子挖掘、神秘蛋孵化进度

## 阶段二：建筑系统扩展

- [x] 3. 现有建筑调整
  - [x] 3.1 灯塔改为事件触发：点亮后永久揭开视野+3范围迷雾，偏向水域旁放置
  - [x] 3.2 营地新增：解除中毒/流血debuff，概率触发交易事件（金币换道具），增加数量
  - [x] 3.3 城市交易事件改为消耗金币，增加数量
  - [x] 3.4 传送门解除地形限制（包括水域）
  - [x] 3.5 农田取消休息加HP，镰刀+农田=+20HP，农田至少连续3格（MapGenerator集群放置）
  - [x] 3.6 教堂改为祈祷事件：-1AP解除诅咒+流血

- [x] 4. 新增建筑（第一批）
  - [x] 4.1 瞭望塔(watchtower)：-1AP，永久揭开视野+3范围迷雾，极小概率道具/小概率金币
  - [x] 4.2 暗礁(reef)：水域专属，战斗/搜索/修船事件
  - [x] 4.3 训练场(training_ground)：一次性，HP上限+10 或 AP上限+1 二选一
  - [x] 4.4 祭坛(altar)：供奉HP换道具（10→common，30→rare，60→epic）或供奉道具换金币
  - [x] 4.5 泉水(spring)：被动回复AP（随机2-5，极小概率回满），不回HP

- [x] 5. 新增建筑（第二批）
  - [x] 5.1 许愿池(wishing_well)：三档投币（10g/50g/100g），各有不同概率奖励
  - [x] 5.2 电话亭(phone_booth)：随机拨号-1AP揭雾，投币20-50g解除诅咒
  - [x] 5.3 食物餐车(food_truck)：20-40g随机，回满AP，-10HP（水杯减半）
  - [x] 5.4 篝火(bonfire)：清除中毒/流血+回满HP，代价重置已触发战斗事件
  - [x] 5.5 树洞(hollow_tree)：森林专属，-3AP随机传送到树洞/泉水，或获得干果
  - [x] 5.6 巨像之手(colossus_hand)：攀爬-2AP视野+5（概率摔伤），冥想+宝石=永久HP上限+5
  - [x] 5.7 自动贩卖机(vending_machine)：10-30g随机，获得随机消耗品，小概率卡住/出双份

- [x] 6. 建筑配置与地图生成
  - [x] 6.1 building.json 新增所有建筑配置（allowedTerrains、triggerEvent、repeatable等）
  - [x] 6.2 MapGenerator 建筑放置逻辑：农田集群、灯塔偏向水域、暗礁水域专属、树洞森林专属
  - [x] 6.3 建筑过夜事件：城市安睡/被抓、营地交易、农田收获
